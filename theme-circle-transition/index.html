实现原理
View Transitions API
浏览器原生过渡接口，在 DOM 更新前后自动捕获快照，无需手动克隆 DOM。
clip-path 圆形
从点击坐标出发，使用 circle() 函数将圆半径从 0 扩展至 170vmax 覆盖全屏。
CSS 自定义属性
通过 CSS 变量统一管理颜色 Token，主题切换只需修改根节点 data-theme 属性。
